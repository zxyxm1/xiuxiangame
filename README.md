# 世界格局
内卷峰（原名：凌霄剑宗）
宗训：“今日不修，明日被修！”
弟子每天打卡炼气12时辰，迟到一次扣10年寿元。
禁地：摸鱼池（其实是上古大能躺平遗址，泡一次+1点“咸鱼值”，全宗通缉）。
拼多多仙盟（原名：万宝阁）
宗门口号：“三人成团，法宝对折！”
卖功法必须拉两个好友入伙，否则价格翻倍。
镇派神功：《砍一刀诀》——每砍一刀，伤害随机×0.1~10倍（可能砍出宇宙大爆炸，也可能只刮掉对方0.1血）。
躺平谷（原名：无为道宫）
表面清静无为，实则全员“躺赚”高手。
核心业务：出租本命元神当WiFi热点（“道友，连我元神热点吗？一灵石/小时，包月送雷劫防护罩”）。

# 修炼体系
境界名称（拒绝高大上，主打一个接地气）：
炼气 → 筑基 → 金丹 → 元婴 → 摸鱼期 → 带薪闭关期 → 飞升（试用期）

炼气期：学会用灵气煮泡面（+1“厨修”属性）。
筑基期：打坐时能自动回复微信（“道友，我正在筑基，稍后回你”）。
金丹期：金丹可当充电宝，给灵器续命（但充多了会爆炸）。
元婴期：元婴会叛逆，经常偷偷玩你的本命飞剑打游戏。
摸鱼期：唯一合法躺平阶段，但天道会派“督修使者”查岗。

# 游戏核心玩法
事件系统：
在“剑道大会”上，对手掏出《拼多多砍一刀诀》，你必须帮他拉两个观众砍价才能继续打。
宗门大比奖励：带薪年假3天（实际只有3时辰）。
属性系统：
传统属性外，加入：
社恐值（越高越难触发社交事件）
摸鱼熟练度（影响躲懒成功率）
凡尔赛指数（影响NPC嫉妒度）
结局分支：
卷王结局：飞升成功，成为天道HR，负责优化其他修士。
咸鱼结局：在躺平谷开“修仙养老院”，收留所有不想飞升的老修士。
暴富结局：靠倒卖“雷劫防护罩”成为仙界首富，但被天道收税到破产。
